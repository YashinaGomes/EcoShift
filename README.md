# 📖 Sobre o Projeto

**Eco Shift** é um jogo **Top-Down** desenvolvido na **Unity**, com foco em sustentabilidade, desperdício e restauração ambiental.

O jogador controla um engenheiro sobrevivente em um mundo degradado pela poluição, lixo e contaminação ambiental. Seu objetivo é coletar peças mecânicas, sobreviver aos perigos do ambiente e restaurar o planeta.

Ao longo da progressão, o mundo evolui visualmente: ambientes poluídos começam a recuperar vida, vegetação e cores.

# 🎮 Gameplay

## Mecânicas Principais

- Movimentação do jogador
- Sistema de coleta
- Sistema de score
- IA de inimigos perseguindo o player
- Barra de HP
- Áreas contaminadas
- Sistema de dano
- Tela de Vitória
- Tela de Game Over
- HUD de gameplay
- Fluxo de cenas com GameManager
